                 │                               │
                 ▼                               ▼
    ┌────────────────────────┐     ┌────────────────────────┐
    │ 1. 查找未同步的 session  │     │ 1. GET /api/sessions   │
    │    (syncedToCloud=false)│     │                        │
    │ 2. 补充 ownerUid       │     │ 2. 过滤掉本地已有的     │
    │ 3. 转换为 SessionDto[] │     │    session (by ID)     │

    │                        │     │    创建 FootballSession│

    └────────────────────────┘     │ 5. 保存到本地数据库     │
                                   │                        │
                                   │ 返回: "已恢复 N 场"     │

## 10. 数据计算公式总览

```
┌─────────────────────────────────────────────────────────────────┐
│                     核心指标计算逻辑                              │
└─────────────────────────────────────────────────────────────────┘

┌──────────────────────────────────────────┐
│ 距离计算                                  │
│ 方法: Haversine 公式 (GPS 经纬度→米)      │
│ 过滤: 精度 > 15m 的点被忽略               │
│ 累加: 相邻有效点之间的距离求和             │
└──────────────────────────────────────────┘

┌──────────────────────────────────────────┐
│ 速度区间                                  │
│                                          │
│  0─────6─────12─────18─────30+ (km/h)    │
│  │ 步行 │ 慢跑 │ 跑步  │ 冲刺  │          │
│  │ 绿色 │ 黄色 │ 橙色  │ 红色  │          │
│  │1.5MET│ 7MET │ 10MET │13+MET│          │
└──────────────────────────────────────────┘

┌──────────────────────────────────────────┐
│ 冲刺检测                                  │
│ 条件: 速度 ≥ 18 km/h 持续               │
│ 计数: 每次从 <18 升到 ≥18 记为一次冲刺    │
└──────────────────────────────────────────┘

┌──────────────────────────────────────────┐
│ 卡路里估算                                │
│                                          │
│ 有心率时 (Karvonen):                      │
│ kcal/min = -55.0969                      │
│          + 0.6309 × HR                   │
│          + 0.1988 × 体重(kg)              │
│          + 0.2017 × 年龄                  │
│                                          │
│ 无心率时 (MET):                           │
│ kcal/min = MET × 体重 × 3.5 / 200       │
└──────────────────────────────────────────┘

┌──────────────────────────────────────────┐
│ 摸鱼指数 (0-100分, 越低越积极)             │
│                                          │
│  ┌──────────────────┬────────┐           │
│  │ 站立时间比 (<0.5km/h) │  35%  │           │
│  │ 低速时间比 (<6km/h)   │  25%  │           │
│  │ 覆盖率缺失 (100%-覆盖) │ 20%  │           │
│  │ 低心率比 (<100bpm)    │  20%  │           │
│  └──────────────────┴────────┘           │
│                                          │
│  0──30──50──70──100                      │
│  拼命  积极  有点  场上                    │
│  三郎  参与  偷懒  观光                    │
└──────────────────────────────────────────┘

┌──────────────────────────────────────────┐
│ 能力雷达 (五维度, 0.0-1.0)                │
│                                          │
│ 速度   = avg(maxSpeedKmh) / 30           │
│ 耐力   = avg(totalDistance) / 10000m     │
│ 强度   = avg(高强度距离/总距离) / 0.3     │
│ 覆盖   = avg(coveragePercent) / 100      │
│ 冲刺   = avg(sprintCount) / 10           │
└──────────────────────────────────────────┘

┌──────────────────────────────────────────┐
│ 覆盖率                                    │
│ 网格: 50行 × 30列 (基于 GPS 边界)         │
│ 计算: 被访问的格子数 / 总格子数 × 100%     │
└──────────────────────────────────────────┘
```
